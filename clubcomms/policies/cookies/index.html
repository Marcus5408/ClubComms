Cookies Policy for ClubComms

Last Updated: 12/10/2023 (December 10th, 2023)

1. Introduction

ClubComms uses certain technologies to enhance user experience. This Cookies Policy explains how we use these technologies, even though the app is not a web application.

2. Local Storage

2.1. ClubComms uses local storage or similar mechanisms to store information on your device. This includes but is not limited to user preferences and authentication tokens for Google OAuth.

2.2. Information stored locally is used solely for the purpose of providing a seamless and personalized experience within the app.

3. Google OAuth

3.1. To enable Google OAuth functionality, ClubComms stores temporary session data locally. This data is used for the authentication process and is not transmitted externally.

4. Opting Out

4.1. This app requires the user to authenticate with Google for access. If you do not wish to use Google OAuth, please do not use the app.
